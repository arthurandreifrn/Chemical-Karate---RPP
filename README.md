Este repositório contém:

Uma pasta contendo os assets dos gráficos do jogo;
Uma pasta contendo as builds do jogo (incluindo uma rom jogável);
E uma pasta contendo os arquivos do projeto no GB Studio.

Download GB Studio: https://chrismaltby.itch.io/gb-studio

Extraia a pasta zip "ChemicalKarate" em um diretório de sua preferência. Para abrir os arquivos no GB Studio, clique em "Abrir" na tela inicial da engine e selecione a pasta "ChemicalKarate". Ao fazê-lo, o programa deve retomar diretamente todo o desenvolvimento ocorrido até então
